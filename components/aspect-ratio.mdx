---
title: "Aspect Ratio: Consistent media sizing in Svelte"
description: "A utility component for Svelte 5 that wraps content and enforces a fixed aspect ratio, keeping images and media consistently sized."
---

The Aspect Ratio component constrains its children to a specific width-to-height ratio. It is commonly used to display images, videos, and other media so that they scale predictably across different screen sizes without layout shifts.

## Installation

```svelte
<script lang="ts">
  import { AspectRatio } from "bits-ui";
</script>
```

## Usage

```svelte
<script lang="ts">
  import { AspectRatio } from "bits-ui";
</script>

<AspectRatio.Root ratio={16 / 9}>
  <img
    src="https://example.com/image.jpg"
    alt="A scenic landscape"
    style="width: 100%; height: 100%; object-fit: cover;"
  />
</AspectRatio.Root>
```

### Common ratios

```svelte
<!-- 16:9 widescreen -->
<AspectRatio.Root ratio={16 / 9}><!-- ... --></AspectRatio.Root>

<!-- 4:3 standard -->
<AspectRatio.Root ratio={4 / 3}><!-- ... --></AspectRatio.Root>

<!-- 1:1 square -->
<AspectRatio.Root ratio={1}><!-- ... --></AspectRatio.Root>
```

### Reusable component

Create a wrapper to avoid repeating boilerplate when using `AspectRatio` for images throughout your application.

```svelte title="MyAspectRatio.svelte"
<script lang="ts">
  import { AspectRatio, type WithoutChildrenOrChild } from "bits-ui";

  let {
    src,
    alt,
    ref = $bindable(null),
    imageRef = $bindable(null),
    ...restProps
  }: WithoutChildrenOrChild<AspectRatio.RootProps> & {
    src: string;
    alt: string;
    imageRef?: HTMLImageElement | null;
  } = $props();
</script>

<AspectRatio.Root {...restProps} bind:ref>
  <img {src} {alt} bind:this={imageRef} />
</AspectRatio.Root>
```

```svelte title="+page.svelte"
<script lang="ts">
  import MyAspectRatio from "$lib/components/MyAspectRatio.svelte";
</script>

<MyAspectRatio
  src="https://example.com/image.jpg"
  alt="An abstract painting"
  ratio={4 / 3}
/>
```

## API Reference

### AspectRatio.Root

The single component of the Aspect Ratio primitive. Wraps its children in a positioned container that enforces the desired ratio.

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `ratio` | `number` | `1` | The desired width-to-height ratio. Pass expressions such as `16 / 9` or `4 / 3` for readability. |

All standard HTML `<div>` attributes are also accepted and forwarded to the underlying element.

## Accessibility

`AspectRatio.Root` renders a plain `<div>` with no additional ARIA roles. Accessibility considerations fall on the content placed inside the component. When wrapping images, always provide a descriptive `alt` attribute; for decorative images, use `alt=""`.
